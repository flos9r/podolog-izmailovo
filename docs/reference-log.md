## ИТОГОВЫЕ ВЫВОДЫ

### Что сейчас в моде (реальные паттерны у сильных сайтов)

1. **Тёплые нейтральные тона** — не белый/синий, а кремовый/зелёный/терракотовый
2. **Serif headings + humanist sans body** — Playfair + систем. шрифт
3. **CTA отделён от nav-ссылок** — бордером, расстоянием, контрастом
4. **Featured article** — первая карточка занимает всю ширину
5. **Numbered steps вместо списков** — «01 / 02 / 03» вместо bullet points
6. **Голос специалиста** — цитата или первое лицо в about-блоке
7. **Success state с конкретным следующим шагом** — «Вас свяжутся через…»
8. **Sticky header с видимым CTA** — всегда доступна кнопка записи
9. **Большие tap targets** — 44×44px минимум на мобильном
10. **Inline breaking news** — полоса над контентом, не попап

### Что уникально (редкие, работающие решения)

1. Нумерованные карточки статей (01/02/03) — встречается редко, усиливает редакционность
2. «Почему регулярный уход» с подзаголовками вместо списка — почти нет в медицинских сайтах
3. Booking flow: «01 → 02 → 03» вместо «доступных слотов» — честнее и удобнее
4. Форма: после отправки — конкретика про следующий шаг (время, дата, стоимость)

### Отброшенные решения

- WebGL/Three.js эффекты — несовместимы с доверием и производительностью
- Агрессивные поп-апы для акций — enshittification паттерн
- Тёмная тема как основная — снижает доверие в healthcare
- Слайдеры как основной layout — плохо на мобильном
- «Доступные слоты» без реальной синхронизации — вводит в заблуждение

### Внедрённые решения

| Паттерн | Источник идеи | Реализация |
|---|---|---|
| CTA отделена бордером от nav | Stripe, Ro Health, Linear | `.nav__cta { border-left: 1px solid var(--color-border) }` |
| Breaking news inline banner | The Guardian, editorial sites | `article-banner` — sticky полоска под header |
| Numbered booking steps | Function Health, Dispatch Health | `booking-info__grid` с 01/02/03 |
| Подзаголовки в «Почему регулярно» | Brightside, Fullscript | `podology__reason` с title + text |
| Success state с конкретикой | The Browser Company, Spruce | «Наталья свяжется и согласует дату, время, стоимость» |
| Featured article (полная ширина) | The Atlantic, editorial sites | `.article-card--featured { grid-column: 1 / -1 }` |
| Numbered article cards | Pentagram, editorial | `.article-card__num` = 01/02/03/04 |
| Burger с min 44×44px | NHS, GOV.UK, Carbon Health | `min-width: 44px; min-height: 44px; padding: 10px` |

## ОТБРОШЕННЫЕ ПАТТЕРНЫ (REJECTED)

| Паттерн | Источник | Причина отказа |
|---|---|---|
| WebGL / Three.js в hero | Lusion, award sites | Performance на mobile, снижает доверие в медицинском контексте |
| Scroll-triggered агрессивные поп-апы | Большинство шаблонов | Enshittification — снижает конверсию и доверие |
| Тёмная тема как основная | Многие SaaS | В healthcare снижает читабельность и доверие |
| Слайдеры как основной layout отзывов | Шаблонные сайты | Плохо на mobile, ограничивает доступность |
| Mega-menu с 10+ ссылками | Корпоративные сайты | Когнитивная перегрузка для простого сайта |
| "Доступные слоты" без синхронизации | Текущий сайт | Вводит пользователя в заблуждение → заменено на шаги |
| 3D стоп / медицинские иллюстрации | Med-tech sites | Неуместно, может пугать |
| Sticky floating CTA поверх контента | E-commerce шаблоны | Перекрывает контент, создаёт visual clutter |
