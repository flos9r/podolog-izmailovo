# Как редактировать сайт — Руководство по контенту

Это практическая инструкция для тех, кто будет вносить правки без глубокого знания кода. Все частые задачи описаны здесь.

---

## 1. Как открыть проект

1. Установите [VS Code](https://code.visualstudio.com/) — бесплатный редактор.
2. Откройте папку проекта: **File → Open Folder** → выберите корневую папку (там лежат `index.html`, `css/`, `js/`, `images/`).
3. Чтобы видеть сайт в браузере:
   - Установите расширение **Live Server** в VS Code → нажмите **Go Live** внизу.
   - Или: `python3 -m http.server 8080` → откройте `http://localhost:8080`.

---

## 2. Структура файлов

```
/
├── index.html              ← Вся разметка сайта
├── css/
│   ├── variables.css       ← Цвета, шрифты, отступы (меняй здесь)
│   ├── styles.css          ← Основные стили
│   └── responsive.css      ← Мобильная версия
├── js/
│   ├── data.js             ← ВСЕ ДАННЫЕ: услуги, отзывы, слоты, статьи, инструменты
│   ├── main.js             ← Логика рендеринга
│   └── form.js             ← Форма записи
├── images/                 ← Все изображения сайта
│   ├── specialist.jpg      ← Фото на hero-блоке (первый экран)
│   ├── about.jpg           ← Фото в разделе "Специалист"
│   └── gallery/            ← Работы (до/после)
│       ├── work-1.jpg
│       └── ...
└── docs/
    └── content-guide.md    ← Этот файл
```

## 3. Где искать секцию в коде

Откройте `index.html` → нажмите **Ctrl+F** → ищите:

| Секция                     | Ищите в HTML         | Данные в         |
|----------------------------|----------------------|------------------|
| Главный экран (hero)       | `id="home"`          | `index.html`     |
| О специалисте              | `id="about"`         | `js/data.js`     |

## 4. Как изменить отзывы

Откройте `js/data.js` → найдите раздел `reviews: [`.

Каждый отзыв выглядит так:
```js
{
  name: "Имя клиента",
  text: "Текст отзыва...",
  rating: 5,
  date: "9 декабря 2025 г.",
  service: "Медицинский педикюр",
  source: "Яндекс Карты",
},
```

**Добавить отзыв:** скопируйте один объект и вставьте выше или ниже, измените данные.

**Удалить отзыв:** удалите весь объект `{ ... },` вместе с запятой.

## 5. Как изменить доступные слоты (availableSlots)

Откройте `js/data.js` → найдите `availableSlots: [`.

```js
{
  date: "2026-04-14",       // дата в формате YYYY-MM-DD
  label: "Пн, 14 апреля",  // отображаемое название
  times: ["10:00", "12:30", "15:00"],  // доступное время
},
```

**Важно:** прошедшие даты фильтруются автоматически — удалять их не обязательно.

## 6. Как вставить фото специалиста

1. Положите файл `specialist.jpg` в папку `images/`.
2. Откройте `index.html` → найдите `hero__photo-placeholder`.
3. Замените блок:
   ```html
   <div class="hero__photo-placeholder" aria-hidden="true">
     <span>Фото специалиста</span>
   </div>
   ```
   На:
   ```html
   <img class="hero__photo" src="images/specialist.jpg" alt="Наталья Сунцова — подолог" />
   ```

## 8. Как добавить фото в галерею

1. Положите файл `work-N.jpg` в папку `images/gallery/`.
2. Откройте `js/data.js` → найдите `gallery: [`.
3. Добавьте новый элемент:
   ```js
   {
     src: "images/gallery/work-7.jpg",
     alt: "Описание результата",
     category: "Педикюр",
   },
   ```

---

## 9. Как изменить навигацию

Откройте `index.html` → найдите `<nav class="nav"`.

Вкладки навигации:
```html
<li><a class="nav__link" href="#about">Специалист</a></li>
<li><a class="nav__link" href="#services">Услуги</a></li>
...
```

Измените текст внутри тега `<a>`. Не меняйте `href` — он указывает на секцию.

---

## 10. Как изменить услуги и цены

Откройте `js/data.js` → найдите `services: [` (описание) и `prices: [` (прайс-лист).

**Услуга:**
```js
{
  id: "medical-pedicure",
  name: "Медицинский педикюр",
  description: "Описание услуги...",
  duration: "от 75 минут до 2 часов",
},
```

**Цена:**
```js
{ service: "Педикюр", price: "от 4 000 ₽", duration: "от 75 мин до 2 часов" },
```

## 13. Как изменить цвета сайта

Откройте `css/variables.css`.

Ключевые переменные:
```css
--color-primary:  #2d5a4a;  /* Тёмно-зелёный */
--color-accent:   #b47b52;  /* Терракота */
--color-bg:       #f7f3ee;  /* Фон страницы */
--color-text:     #1f1a17;  /* Основной текст */
```

---

## 14. Как проверить мобильную версию

1. Откройте сайт в браузере.
2. Нажмите **F12** (или Ctrl+Shift+I на Windows).
3. Нажмите иконку телефона — **Toggle Device Toolbar**.
4. Проверьте ширины: `320px`, `360px`, `375px`, `390px`, `412px`.

---

## 15. Как изменить статьи

Откройте `js/data.js` → найдите `articles: [`.

Каждая статья содержит:
- `title` — заголовок
- `excerpt` — краткое описание (для карточки)
- `content` — массив секций с `heading` и `text`
- `sources` — источники

Чтобы добавить новую статью — скопируйте один объект и заполните.

## Важные правила по формулировкам

- Подолог — **не врач**. Не используйте слова: "лечение", "диагноз", "медицинская помощь", "пациент".
- Используйте: "уход", "обработка", "процедура", "профессиональная помощь", "клиент".
- В конце сайта есть дисклеймер — не удаляйте его.
