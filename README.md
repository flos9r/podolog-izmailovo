# Сайт подолога — Наталья Сунцова, Измайлово

Статический лендинг для подолога. Чистый HTML/CSS/JavaScript, без зависимостей и сборщиков.

---

## Структура проекта

```
/
├── index.html              # Единственная HTML-страница — вся разметка здесь
├── css/
│   ├── variables.css       # CSS-переменные (цвета, шрифты, отступы)
│   ├── styles.css          # Основные стили всех секций
│   └── responsive.css      # Адаптивность (mobile-first, breakpoints)
├── js/
│   ├── data.js             # ВСЕ данные сайта (редактируйте только здесь)
│   ├── main.js             # Рендеринг данных, слайдер, слоты, галерея
│   └── form.js             # Валидация и отправка формы записи
├── images/
│   ├── specialist.jpg      # (добавить) Фото специалиста — hero-секция
│   ├── about.jpg           # (добавить) Фото для раздела "Специалист"
│   └── gallery/
│       ├── work-1.jpg      # (добавить) Фотографии работ (до 6 штук)
│       └── ...
└── README.md
```

## Как открыть проект локально

1. Скачайте или склонируйте репозиторий на компьютер.
2. Откройте папку проекта в любом редакторе (VS Code, WebStorm, Sublime Text).
3. Чтобы открыть сайт в браузере, воспользуйтесь любым из способов:
   - Установите расширение **Live Server** (VS Code) и нажмите "Go Live".
   - Или запустите локальный сервер командой: `python3 -m http.server 8080` — затем откройте `http://localhost:8080`.
   - Или просто откройте файл `index.html` в браузере двойным кликом (большинство функций работает).

## Как добавить фотографии

### Фото специалиста (hero-блок)

1. Положите файл в папку `images/` с именем `specialist.jpg`.
2. Откройте `index.html` и найдите (Ctrl+F): `hero__photo-placeholder`.
3. Замените `<div class="hero__photo-placeholder" ...>...</div>` на:
   ```html
   <img class="hero__photo" src="images/specialist.jpg"
        alt="Наталья Сунцова — подолог в Измайлово"
        width="400" height="500" />
   ```

### Фото в разделе "Специалист"

1. Положите файл в `images/about.jpg`.
2. В `index.html` найдите: `about__photo-placeholder`.
3. Замените `<div class="about__photo-placeholder" ...>...</div>` на:
   ```html
   <img class="about__photo" src="images/about.jpg"
        alt="Наталья Сунцова — подолог в рабочем кабинете"
        width="600" height="750" />
   ```

### Фотографии работ (галерея)

1. Положите файлы в `images/gallery/work-1.jpg`, `work-2.jpg`, ..., `work-6.jpg`.
2. Откройте `js/data.js` и найдите массив `gallery:`.
3. Для каждого изображения укажите правильный путь в поле `src`.

Все изображения на сайте защищены от выхода за контейнер через `max-width: 100%` и `object-fit: cover`.
Для зарезервированного места под изображение используйте CSS-класс `.img-wrap` с модификатором:
- `.img-wrap--portrait` — соотношение 4:5 (портрет)
- `.img-wrap--banner` — соотношение 16:9 (горизонтальный баннер)
- `.img-wrap--square` — квадрат 1:1

## Как изменить доступные записи (расписание)

Откройте файл `js/data.js` и найдите массив `availableSlots:`.

Каждый элемент массива — это один день:

```javascript
availableSlots: [
  {
    date:  "2026-04-01",        // дата в формате YYYY-MM-DD
    label: "Вт, 1 апреля",     // как будет показано пользователю
    times: ["10:00", "12:00", "16:00"],  // доступные слоты этого дня
  },
  // добавьте следующий день ниже...
]
```

Что нужно знать:
- Прошедшие даты скрываются автоматически — удалять их необязательно.
- Выбранное время автоматически подставляется в форму записи.
- После изменения файла достаточно обновить страницу браузера.

## Как проверить мобильную версию

Способ 1 — инструменты разработчика браузера:
1. Откройте сайт в Chrome или Firefox.
2. Нажмите F12 → включите режим устройства (иконка телефона или Ctrl+Shift+M).
3. Выберите разрешение: 320px, 360px, 390px или 412px.
4. Прокрутите страницу — убедитесь, что нет горизонтального скролла.

Способ 2 — реальное устройство:
1. Убедитесь, что телефон и компьютер в одной Wi-Fi сети.
2. Запустите `python3 -m http.server 8080`.
3. Узнайте IP компьютера (`ipconfig` / `ip a`) и откройте `http://192.168.x.x:8080` на телефоне.

## Как изменить цвета и дизайн

Все основные параметры дизайна находятся в `css/variables.css`:

```css
--color-primary:   #7b6e8a;  /* основной акцент (лиловый) */
--color-accent:    #c9a87c;  /* второй акцент (золотой) */
--color-bg:        #faf9f7;  /* фон страницы */
--color-bg-dark:   #3d3540;  /* тёмный фон (hero, footer) */
--font-heading:    'Playfair Display', serif;
--font-body:       'Inter', sans-serif;
```

## Как изменить контент (тексты, цены, контакты)

Весь контент хранится в `js/data.js`. Редактируйте объект `siteData`:

- `specialist` — имя, должность, описание, сертификаты
- `services` — список услуг с описаниями
- `prices` — прайс-лист
- `gallery` — пути к фото работ
- `reviews` — отзывы (имя, дата, рейтинг, текст)
- `advantages` — блок "Почему наш кабинет"
- `contacts` — телефон, email, адрес, мессенджеры
- `availableSlots` — доступные даты и время для записи
